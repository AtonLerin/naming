Naming
======
`naming` is a pure python naming convention manager.

Dependencies
------------
- [nose](http://nose.readthedocs.org/en/latest) (unit testing)
- [wishlib](http://github.com/csaez/wishlib)

Installation
------------
Clone the repo and type in a terminal:

    python setup.py install
